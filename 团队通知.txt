📋 快速开始（获得权限后）：
1. 克隆项目到本地：
   git clone https://github.com/ethan7zhanghx/ai-calculator.git

2. 进入项目目录并安装依赖：
   cd ai-calculator
   npm install --legacy-peer-deps

3. 启动开发服务器：
   npm run dev

4. 在浏览器打开：http://localhost:3000
